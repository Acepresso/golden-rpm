# golden-rpm

Trivial definition of a multi-arch RPM build (amd64/arm64) in compliance with Conforma
policy.

The latest released image is available at `quay.io/konflux-ci/ec-golden-rpm:latest`.

The image is signed and attested by Tekton Chains. For verification, use the in-cluster public key:
`k8s://openshift-pipelines/public-key`.
